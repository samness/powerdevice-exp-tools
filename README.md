# 发电设备安全测试工具 V0.1 beta

## 版本信息
- 版本号：V0.1 beta
- 发布日期：2025-03-28
- 构建号：20250328.1

## 功能特性
- 支持多种发电设备漏洞扫描
- 支持特定设备漏洞利用
- 图形化界面操作
- 实时日志显示
- 漏洞详情展示

## 支持的设备
- 西门子SGT-800
- GE LM6000
- 三菱M701F
- GE 9HA.02
- 西门子SGT5-8000H
- 三菱电力M701JAC
- GE HA-Predix
- 西门子SGT-6000

## 安装说明
1. 确保已安装Python 3.8或更高版本
2. 安装依赖包：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行程序：
   ```bash
   python power_security_tester.py
   ```

## 使用说明
1. 选择目标设备类型
2. 选择具体设备型号
3. 选择要测试的漏洞
4. 输入目标IP地址和端口
5. 点击"开始测试"按钮

## 注意事项
- 本工具仅用于安全测试和评估
- 使用前请确保获得测试授权
- 建议在测试环境中使用
